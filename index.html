Yosemite National Park Visitors Map

Project Description

This is a map designed as a replacement to the current visitors map for Yosemite National Park. The existing park visitors map emphasizes the roads, campgrounds, and other features of teh built environment in the park, while pushing the terrain and the natural beauty of teh park to the very back of the visual hierarchy. The map that I made aims to promote the beauty and the possibility of exploration to park visitors, through the use of shaded relief and hyper realistic land cover, while maintaining the information now available on the map map, such as the locations of important park services.

Tools Used

  • ArcGIS
  • QGIS
  • Adobe Photoshop
  • Adobe Illustrator
  • Blender

Misguided and Dangerous

Project Description

This is a project that examines the coverage of the Syria refugee crisis in the New York Times. While the Times covers the situtation in some depth, its coverage is not correlated to the number of refugees arriving in each country, a trend that spans both space and time. This project examines this lack of statistical correlation, and hypothesizes that this could be a potential cause of a bias among Americans responsible for their unwillingness to accept Syrian refugees.

Tools Used

  • ArcGIS
  • Adobe Photoshop
  • Adobe Illustrator
  • Python

Tour of Scandinavia Promotional Poster

Project Description

This is a map I made for my Cartographic Graphic Design class. The map promotes a fictional bike race through Scandinavia, modeled on the well-known Tour De France stage race. Its visual hierarchy is dominated by the race course and the cities in which the stages start and stop. My favorite part of the map, though, is the shaded relief basemap of the region, which is a beautiful representation of the terrain, and highlights the topography of Sweden, Norway, and Denmark.

Tools Used

  • ArcGIS
  • Adobe Photoshop
  • Adobe Illustrator
  • Terrain Sculptor

Valley Fire Infographic

Project Description

This is an infographic I designed as the Valley Fire burned in Lake County, Calif. in late September of 2015. The magnitude and suddenness of the destruction made it an interesting topic to try to move beyond the numbers of my other fire modeling projects and try to really get a sense of who is being affected by these massive wildfires in the west. In this infographic, I look at several economic demographics that I feel are important in understanding the plight of the victims of the fire.

Tools Used

  • ArcGIS
  • Adobe Illustrator

Firesphere

Project Description

Firesphere is a project I founded to improve the communication and dissemination of fire risk data within fire prone areas in California. It is a two part, community centered project that features both a mobile application for the iOS platform and a web-portal for data exploration and visualization. Community residents are encouraged to submit observations of fuel type in their surroundings via the iPhone app. These crowdsourced data points are used to calculate high spatial and temporal resolution, which are then published to the web portal for community consumption.

Tools Used

  • Titanium Mobile Front-End
  • Javascript Web Front-End
  • PHP Backend Web Services / API
  • Python Backend Risk Calculations
  • ArcGIS Spatial Data Server
